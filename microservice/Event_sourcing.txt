Event sourcing 的定义，要记住的三句说话

Event Sourcing is an alternative way to persist data.  | Event sourcing 是另类的保存数据的方法
traditional state-oriented application only keeps the latest version of the entity state, | 相对于之保留最终状态的常规方法
Event Sourcing stores each state change as a separate record called an event. | 每次变化，event sourcing 都记录为 event
